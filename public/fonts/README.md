# Licensed Aeonik (optional)

If you have **Aeonik** desktop/web license files from CoType Foundry, add the `.woff2` files here (for example `Aeonik-Regular.woff2`, `Aeonik-Medium.woff2`, `Aeonik-Bold.woff2`) and switch `app/layout.tsx` to load them with `next/font/local` instead of the current Google-hosted stand-in.

The CSS variable name stays `--font-aeonik` so the rest of the site does not need to change.
